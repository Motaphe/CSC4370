A beautiful music review application with Catppuccin theme, built with Express.js and MySQL.

## Quick Start

```bash
# Start the application
docker-compose up --build

# Access at http://localhost:3000
```

## Features

- **Add Reviews**: Complete form for music reviews with 5-star rating
- **Browse Reviews**: Select artists and view all their reviews
- **Beautiful UI**: Catppuccin-themed dark interface
- **Responsive Design**: Works on mobile and desktop

## Database

- **MySQL 8.0** with sample data (Beatles, Pink Floyd, Led Zeppelin)
- **Auto-initialization** with schema and sample reviews
- **Persistent storage** with Docker volumes
